## 日本 VPS 的几条"命脉"：线路决定一切

很多人选日本 VPS 第一眼看的是价格，其实第一眼应该看的是**线路**。同样是"日本机房"，走不同线路，体验能差出一个数量级。

简单科普下你会在 VMISS 日本套餐里反复看到这几个词：

- **IIJ 线路**：日本本土最大的 ISP 之一，国际出口质量稳定，适合做海外业务、面向全球用户的站点，回国体验中等偏上，性价比高。
- **BGP 线路**：多线 BGP 接入，自动选路，回程通常走 IIJ 或软银等组合，对国内三网都有覆盖，综合表现稳。
- **三网优化（TRI）**：电信走 CN2、联通走 AS10099+AS4837、移动走 AS58453+AS9808，三网回程全部直连优化，是国内访问体验最好的一档，但也是最贵的一档。
- **CN2 GIA**：电信高端线路，延迟低、稳定性强，电信用户最爱。
- **软银 SoftBank**：联通用户回程快，适合联通占比高的场景。

一句话总结：**预算紧、做海外业务 → IIJ；要稳、要平衡 → BGP；只服务国内用户、要快 → 三网优化（TRI）。** 这条逻辑后面我会反复用到。
